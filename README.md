# ToDoList
 feature-rich to-do list web app built with Node.js, Express, MongoDB, and Mongoose. Create custom lists, add, complete, and delete items. User-friendly interface with dynamic rendering using EJS. Enjoy efficient task management with this powerful application.

## Live Demo

Check out the live version of the ToDoList web app [here](https://tanvir-todolist-s394.onrender.com).

## Features

- Add new tasks with a title and optional description.
- Edit existing tasks to update their details.
- Mark tasks as complete or incomplete.
- Delete tasks that are no longer needed.
- Filter tasks by status (all, complete, incomplete).
- Responsive design for seamless usage on different devices.

## Technologies Used

- Front-end: HTML, CSS, JavaScript
- Back-end: Node.js, Express.js
- Database: MongoDB
